Rate My Professor Web Scrapper

Python script that uses requests and BeautifulSoup4 python libaries to allow users to scrape search results from ratemyprofessor.com.

Given a name to search for, the program attempts to find data on ratemyprofessor.com for any professor at the University of Michigan that matches the given name. Results are based on the results that ratemyprofessor.com would give if you searched on the site directly using the same name.

Users are prompted to input a search name and the script will page through all the results to find any University of Michigan professor returned by the search.
