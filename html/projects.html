RateMyProfessor Web Scrapper

While I value the information on RateMyProfessor.com, I find the user interface very cumbersome. I wanted to specific information about professors at the University of Michigan, so I wrote a program to give me only that information. Written in Python, the program allows you to search RateMyProfessor.com and displays the "Overall Quality", "Would Take Again", and "Level of Difficulty" scores for any professor that was listed as a University of Michigan, Ann Arbor professor in the search results.
